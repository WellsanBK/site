Qual é o foco?

O foco será testar e aprimorar os comandos do html em busca de criar um site consistente. Após isso, aprimorar o ,CSS e por fim o javascript.


Lista de tags

  1. <ol> … </ol> Lista ordenada
  2. <ul> … </ul> Lista sem ordem
  3. <li> … </li> Itens de uma lista. Podem ser utilizado tanto com as tags <ol> quanto <ul>
  4. <ol> type=”?”> Define o tipo de ordenação: A, a, I, i, 1
  5. <ol> start=”??”> Define o valor de início da lista.
  6. <ul> type=”?”> Define o tipo de lista não-ordenada: disc, circle, square
  7. <value=”??”> Define o valor do item
  8. <li> type=”??”> Define o tipo do item
  9. <dl> … </dl> Lista de conceitos
  10. <dt> … </dt> Termo ou frase
  11. <dd> … </dd> Definição detalhada sobre o item
